🧠 Ollama Flask Web App

This project demonstrates how to run a local AI chatbot using Ollama as your language model, Flask for the backend, and a simple HTML/JavaScript frontend. It’s ideal for local development, secure tunneling, and scalable cloud deployment.

📦 Features

  • Run Ollama locally for privacy-first AI
  • Lightweight Flask backend
  • HTML/JS frontend UI
  • Secure Cloudflare Tunnel for HTTPS access
  • Dockerfile for cloud migration

🚀 Get Started

git clone https://github.com/CptNope/Ollama-Flask-Web-App
cd Ollama-Flask-Web-App
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
ollama run llama3
python app.py

🌍 Secure Public Access

Use Cloudflare Tunnel:

cloudflared tunnel --url http://localhost:5000

🐳 Deploy in the Cloud

docker build -t ollama-flask-app .
docker run -p 5000:5000 ollama-flask-app
